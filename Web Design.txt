Headings
HTML has 6 levels of heading <h1> to <h6>
<h1> This is main heading </h1>
<h2> This is level 2 heading </h2>
     (    Text of heading   )

Paragraph
<p> Paragraph written in here (can be more than 1 sentence) </p>
(For new line, just </p> then start a new paragraph)
Or for new line, can use line break <br/>
Line break with a line <hr/>

<b> Bold </b> and <i> Italic </i>

<b> Bold
<strong> Important text
<i> Italic
<em> Emphasized text
<mark> Marked text
<small> Small text
<del> Deleted text
<ins> Inserted text
<sub> Subscript
<sup> Superscript
<!==> Comments

Lists

Link (id)
<a href="#top"> JUMP TO TOP </a>
But have to assign what is #top
eg. <h1 id="top"> This is the top </h1> (This will be "top")

Tables
<table> - HTML table
<tr> - Each table row
<th> - Table header (by default, table headings are bold and centered
<td> - table data/cell
<th scope="row/col"> _____ </th> - row or column heading
<td colspan="2"> ChickenMcNuggests </td> - spans 2 columns

eg.
<table>

<tr>
<td> 1 </td>
<td> 2 </td>
<td> 3 </td>			Table
</tr>				1 2 3
				4 5 6

</table>
